🧀 Site de Laticínio – Projeto Educacional



📌 Sobre o Projeto



Este projeto consiste no desenvolvimento de um site institucional de um laticínio existente, criado exclusivamente para fins educacionais.



O site não é oficial e não possui qualquer vínculo comercial ou institucional com a empresa real.

Seu desenvolvimento teve como objetivo praticar e demonstrar conhecimentos em desenvolvimento web, simulando um site real do setor de laticínios.



🎯 Objetivo Educacional



Praticar HTML5 e CSS3



Desenvolver um site institucional



Trabalhar com estrutura e organização de páginas



Aplicar conceitos de design e layout



Simular um projeto real para aprendizado



⚠️ Aviso:

Este site foi criado apenas para fins educacionais. Apesar de se basear em uma empresa real, ele não é um site oficial e não deve ser utilizado para fins comerciais.



🛠️ Tecnologias Utilizadas



HTML5



CSS3



Font Awesome



Estrutura de layout responsivo
